# Promptathon Judge - GitHub Pages

This folder is ready to publish with GitHub Pages.

Important:
- GitHub Pages is static only.
- Workspace import/export and manual judging work in the browser.
- Live sync and GPT-5.4-mini API judging require the Node backend/Azure hosting, not GitHub Pages alone.

Publish option:
1. Upload `index.html` to the root of your GitHub Pages repository, or to `/docs` if your repo is configured to publish from docs.
2. Enable GitHub Pages in repository Settings > Pages.
